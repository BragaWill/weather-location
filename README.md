## react app to start development

## Run app:

set the file .env with your wearth API key `REACT_APP_OPEN_WEATHER_KEY=<<API-key>>`

- https://openweathermap.org/api

npm start

## Run tests:

npm test

tools :

- mochajs , chaijs, sinonjs and enzymejs.

## Run build:

npm run build

## Run ESlint:

npm run eslint
